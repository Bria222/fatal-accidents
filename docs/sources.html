Data Sources

Complete documentation of all datasets used in the analysis

fatal_accidents_drivers.csv

Contains detailed records of all fatal driver accidents in Formula 1 history.

Columns:
  • Driver Name
  • Age
  • Team / Constructor
  • Circuit
  • Country
  • Date of Accident
  • Cause of Accident
Source:

Compiled from Motorsport Memorial, FIA reports, Wikipedia archives, and race historical datasets.

fatal_accidents_marshalls.csv

Records of fatal accidents involving race marshalls and track officials.

Columns:
  • Name
  • Role
  • Circuit
  • Country
  • Year
  • Description of Incident
Source:

Motorsport Memorial, FIA safety documentation, official race archives.

red_flags.csv

Dataset containing all race red flag events and interruptions.

Columns:
  • Race
  • Year
  • Lap
  • Reason for Red Flag
  • Duration
  • Restart Type
Source:

FIA race logs, Formula1.com event summaries, Ergast API.

safety_cars.csv

Data describing every safety car deployment in Formula 1.

Columns:
  • Race
  • Year
  • Lap
  • Reason
  • Duration
  • Safety Car Model
Source:

Formula1.com archives, FIA race director notes, verified motorsport databases.

Data Cleaning & Preparation

Cleaning Steps Applied:

  • Removed duplicate records across datasets
  • Standardized date formats (YYYY-MM-DD)
  • Normalized circuit and country names for consistency
  • Corrected known historical inaccuracies using FIA documentation
  • Handled missing values via manual lookup or removal
  • Merged driver and track information for richer analysis
  • Converted categorical fields (e.g., cause of accident) into analysis-ready formats
