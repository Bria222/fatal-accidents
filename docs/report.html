Report Overview

This report presents a full analytical exploration of fatal accidents in Formula 1, combining historical datasets of drivers, marshalls, red flag events, and safety car deployments.

The analysis was conducted in R using RMarkdown, including data cleaning, statistical analysis, visualizations, and final insights in a reproducible workflow.
